13 Security Modules

Security Headers

Checks 14 critical HTTP security headers including CSP, HSTS, and X-Frame-Options.

Port Scanning

Scans 25+ common ports for services like SSH, FTP, database servers, and admin panels.

SQL Injection

Tests 20+ payloads including union-based, error-based, blind, and time-based SQLi.

XSS Detection

25+ XSS payloads tested across multiple parameters — script, event, SVG, and polyglot.

SSL/TLS Analysis

Certificate validity, expiry, weak protocols (TLS 1.0/1.1), and cipher configuration.

CORS Checker

Tests for wildcard origins, reflected origins, null origin, and credential leaks.

Cookie Security

Checks HttpOnly, Secure, SameSite flags on all cookies with session awareness.

Tech Fingerprinting

Identifies server, CMS, JS frameworks, programming languages from headers and source.

HTTP Methods

Tests for dangerous methods: PUT, DELETE, TRACE, CONNECT — flags XST and file upload risks.

Open Redirects

Tests 25+ redirect parameters with bypass payloads for phishing and token theft.

Info Disclosure

Detects leaked headers, HTML comments, stack traces, debug endpoints, and source maps.

Clickjacking

Tests X-Frame-Options and CSP frame-ancestors for UI redressing protection.

Attack Report

For every finding: attack vector, impact, step-by-step exploitation, severity, and remediation.
